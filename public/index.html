meet new friends

Chibi is a SMS social networking service that helps you find new friends. Chibi is 100% anonymous, works on all phones and does not require an Internet connection. Just text to register and start chatting immediately.

How it works

  • Registration & Chat

    After a simple one message registration process, a new user is instantly matched and placed in an anonymous chat with an existing user. Both users can continue to chat with each other, exchange details or end the session and search for a new match.

  • Market Opportunity

    With no online registration, credit card, smartphone or Internet connection required, Chibi lowers the barrier to entry enabling it to tap into new and emerging markets.

  • Business Model

    Chibi generates revenue via premium short code SMS. With a 50% revenue sharing deal with the telcos, Chibi generates cash from each user generated text message. Chibi also receives a free system generated text message for each user generated message which enables it to create the unique interactive chat environment.
